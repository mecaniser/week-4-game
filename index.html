    • Using this four crystals:
    • Blue

  2. Each Crystal has its own number which is randomly picked between 1-12
  3. Click on each one will add points to total score.
  4. You can win only if your number will match the random number picked at the begining of the game.
  5. If your total score is more than the number picked, you lose.
  6. When a new game starts, your crystal number will be different and hidden!
